rotational_sort module

rotational_sort.rotational_sort(points, ids=None, ccw=False, centre=None)

Given a list of x, y points, sort by angle around a midpoint.

Parameters

  • points (list[(x,y)]) – list of (x, y) points to be sorted

  • ids (list, optional) – this list will be sorted identically to points

  • ccw (bool) – sort counter-clockwise if True

  • centre (tuple(x,y), optional) – define a set centre to sort around

Returns

sorted list of points, list of IDs sorted identically

Return type

list[(x, y)], list
